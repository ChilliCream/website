React Rasta 1.0.0

May 3, 2018

Rafael Staib

Today we proudly released the first version of React Rasta.

React Rasta is a responsive 12 column grid system which offers a lot of features and flexibility. Its API is clean and simple to ease development. React Rasta is also well tested to guarantee high quality. We hope React Rasta will support you by creating great application. Have fun!
